This sample site showcases a Single Page Application (SPA) Web Application built on the latest Angular framework. It is deployed to Amazon Simple Storage Service (S3) and is served via Amazon CloudFront.

The application is integrated with Okta's OIDC idenity provider for authentication and authorization.

The dynamic data is retrieved and updated by calling a RESTful Web Services coded in C# using .NET Core framework. The web services are containerized and served via AWS Fargate clusters running on Amazon Elastic Container Service (ECS). The Web Services are also configured to require users to be authenicated prior to returning the data.

The data is stored in a PostgreSQL database running on Amazon Aurora.
